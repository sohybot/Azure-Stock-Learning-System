03 · Technical Challenges

기술 챌린지 &
해결 전략

3가지 핵심 기술 문제를 직접 정의하고 해결하여 상용화 가능한 아키텍처를 설계했습니다.

CHALLENGE 01

LLM API 비용 폭증

500명 에이전트가 매 틱마다 뉴스를 분석하며 Azure OpenAI 토큰 비용이 급증하는 문제가 발생했습니다.

해결 전략

메타 휴리스틱 기반 워크플로우 최적화 (GA + SA), 프롬프트 압축률 최적화, 동적 라우팅 구조 설계, 자가 검증 노드 배치 구조 개선

API 비용 49.5% 절감 · 에이전트 지능 점수 93.4 유지
CHALLENGE 02

Human-in-the-Loop 병목

AI 주문 대기열로 인한 사용자 체결 지연이 발생해 체감 UX가 크게 저하되었습니다.

해결 전략

Market Maker 기반 동적 유동성 공급 로직 설계, 정상 호가 범위 내 즉시 체결 구조 구현

사용자 주문 1초 이내 체결 · 체감 UX 대폭 개선
CHALLENGE 03

DB 과부하 & 성능 저하

trades 테이블 수십만 건 누적으로 Full Table Scan이 발생해 쿼리 성능이 급격히 저하되었습니다.

해결 전략

Timestamp 인덱싱 적용, 거래량 계산 로직 비동기 분리, 메모리 플러시 스크립트 설계

쿼리 성능 최적화 완료
